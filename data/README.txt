Coloque o arquivo de veiculos aqui com o nome:
  veiculos.xlsx

O sistema lera automaticamente esse arquivo sempre que a pagina Checklist for carregada.
Para atualizar os dados, basta substituir o arquivo por uma versao nova com o mesmo nome.

O sistema tambem aceita o arquivo em:
  flask_ativuz/uploads/VEICULOS POR CONTRATO DE LOCACAO.xlsx
(caminho alternativo — a pasta data/veiculos.xlsx tem prioridade)
